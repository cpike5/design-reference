# Design Reference Collection

This directory contains a curated collection of HTML design prototypes demonstrating various visual styles, design patterns, and aesthetic approaches. Each prototype is a fully implemented HTML/CSS design that can serve as inspiration and reference for .NET web applications.

## About This Collection

These prototypes showcase different design languages suitable for various business contexts and content types. They range from minimalist professional interfaces to rich editorial designs, from modern gaming aesthetics to vintage-inspired layouts.

## Usage Guidelines

### Selecting a Prototype

1. **Match the Mood:** Choose based on emotional tone needed for your content
2. **Consider Your Audience:** Technical users vs. general public vs. creative professionals
3. **Content Type:** Data-heavy vs. editorial vs. visual
4. **Brand Alignment:** Modern vs. traditional vs. luxury positioning

### Extracting Design Tokens

Each prototype contains CSS custom properties (variables) that can be extracted:

```css
:root {
    --primary-color: #value;
    --background: #value;
    --text-color: #value;
    --font-family: 'Font Name', fallback;
}
```

### Responsive Considerations

Most prototypes include responsive breakpoints. Check the `@media` queries in each file for mobile adaptation strategies.

### Accessibility Notes

- All prototypes use semantic HTML
- Most include focus states for keyboard navigation
- Consider adding ARIA labels for complex interactive elements
- Check color contrast ratios for your specific brand colors

## Design System Integration

These prototypes can serve as the foundation for a Blazor-based design system:

1. **Extract Design Tokens:** Copy CSS variables to a centralized tokens file
2. **Component Mapping:** Map HTML patterns to Blazor components
3. **Theme Variants:** Use multiple prototypes to create theme variations
4. **Icon Systems:** Note icon usage patterns (Hero Icons, Lucide, SVG inline)
5. **Animation Patterns:** Document transition and animation approaches
