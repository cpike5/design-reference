Accordion / Collapsible

What is this component?

This is an accordion component that allows you to expand and collapse sections. Click the header to toggle the content visibility.

How does the animation work?

The animation uses CSS transitions with max-height and opacity changes. The active class controls the expansion and rotation of the icon.

Can multiple sections be open?

Yes, in this implementation multiple sections can be open at the same time. If you prefer only one open at a time, you can modify the JavaScript to close others.
