# design-reference

A curated collection of HTML/CSS design prototypes showcasing various visual styles and design patterns. Each prototype is a standalone HTML file that can serve as inspiration or reference for web applications.

## Quick Start

Open any prototype directly in a browser:

```bash
# Open a prototype
start prototypes/discord-dashboard.html
```

## Tech Stack

- Plain HTML/CSS (no build step)
- Tailwind CSS via CDN (some prototypes)
- Google Fonts
- Lucide/Hero icons (inline SVG)
